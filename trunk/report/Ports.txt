Input Ports:
0x01 -> K0 (bit 0 do valor do teclado)
0x02 -> K1 (bit 1 do valor do teclado)
0x04 -> K2 (bit 2 do valor do teclado)
0x08 -> K3 (bit 0 do valor do teclado)
0x10 -> Not Defined (Yet)
0x20 -> Not Defined (Yet)
0x40 -> K_Val (Flag de Valor Válido do Teclado)
0x80 -> RxRdy (Flag que indica que o LCD está apto a escrever)

Output Ports:
0x01 -> RxD (Dados Enviados para a Pal que processa os dados para o LCD)
0x02 -> RxC (Dados Enviados para a Pal que processa os dados para o LCD)
0x04 -> Not Defined (Yet)
0x08 -> Not Defined (Yet)
0x10 -> Not Defined (Yet)
0x20 -> Not Defined (Yet)
0x40 -> Not Defined (Yet)
0x80 -> Not Defined (Yet)
